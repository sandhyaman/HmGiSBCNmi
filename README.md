# Python协同过滤儿童图书推荐系统

## 核心功能
- 用户登录注册
- 图书推荐
- 热销图书展示
- 儿童图书分类
- 公告信息发布
- 图书分类浏览
- 图书详情展示
- 评论点赞功能
- 加入购物车
- 下单功能
- 用户管理
- 图书管理
- 订单管理

## 适用场景
- 儿童图书电商平台
- 学校图书馆
- 儿童阅读平台
- 图书馆自动化系统

## 技术特点
- 使用 Django 作为后端框架，实现用户管理和图书管理功能。
- 利用 MySQL 数据库存储图书、用户和订单信息。
- 前端采用 Vue.js，提供用户友好的交互界面。
- 基于协同过滤算法实现个性化图书推荐。
